# SofIA Frontend

Interface web do assistente **sofIA** by SynastrIA Networks.

São 4 IAs diferentes, somando 1,22 trilhão de parâmetros no total. Um router inteligente avalia a complexidade da pergunta e usa apenas uma delas por vez — de 32B até 120B.

## Stack

- React + TypeScript + Vite
- Tailwind CSS + shadcn/ui

## Rodando localmente

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
